## **Project Overview**

This project replicates the **Facebook signup page layout**. Users can input their **first name, surname, mobile number, password, date of birth, and gender**. Form validation ensures that all required fields are correctly filled before submission.

---

## **Features**

1. Responsive **signup form layout**  
2. **Input validation** using jQuery  
3. Supports **first name, last name, mobile number, password, DOB, and gender selection**  
4. Styled with **CSS** to resemble Facebook’s signup form  
5. **Links** for terms and login page included  

---

## **Technologies Used**

1. **HTML5** – Structure of the signup form  
2. **CSS3** – Styling, layout, and responsive design  
3. **JavaScript (jQuery)** – Client-side validation  
4. **jQuery Validation Plugin** – For form validation
