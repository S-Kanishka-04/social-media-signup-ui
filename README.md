A simple **Facebook-style signup form** created using **HTML, CSS, and JavaScript (jQuery Validation)**. This project demonstrates form layout, styling, and client-side form validation.

## **Project Overview**

This project replicates the **Facebook signup page layout**. Users can input their **first name, surname, mobile number, password, date of birth, and gender**. Form validation ensures that all required fields are correctly filled before submission.

---

## **Features**

1. Responsive **signup form layout**  
2. **Input validation** using jQuery  
3. Supports **first name, last name, mobile number, password, DOB, and gender selection**  
4. Styled with **CSS** to resemble Facebook’s signup form  
5. **Links** for terms and login page included  

---

## **Technologies Used**

1. **HTML5** – Structure of the signup form  
2. **CSS3** – Styling, layout, and responsive design  
3. **JavaScript (jQuery)** – Client-side validation  
4. **jQuery Validation Plugin** – For form validation  

---

## **Installation**

1. Clone the repository:

```bash
git clone https://github.com/yourusername/facebook-signup-clone.git
```
2. Open the index.html file in a browser.


## **Usage**

1. Fill out the signup form fields:
   - First Name
   - Surname
   - Mobile Number
   - Password
   - Date of Birth
   - Gender

2. Click **Sign-up**.

3. The form will validate the required fields before submission.
3. Ensure you have an internet connection to load Google Fonts and jQuery CDN.

## **Form Validation Rules**

| Field        | Rules                                | Error Message Example                           |
|--------------|-------------------------------------|------------------------------------------------|
| First Name   | Required, minimum 5 characters      | "please enter minimum 5 letters for first name" |
| Surname      | Required, minimum 5 characters      | "please enter minimum 5 letters for surname" |
| Mobile       | Required                            | "please enter mobile number"                  |
| Password     | Required, minimum 8 characters      | "please enter minimum 8 characters"          |
| DOB          | Day, Month, Year required           | "please select your date of birth"           |
| Gender       | Required                            | "please select your gender"                  |
